# Saúde & Cia - Sistema de Gestão Clínica

Este projeto consiste na implementação de um sistema de gerenciamento para a clínica médica Saúde & Cia, desenvolvido como parte da disciplina de Programação Orientada a Objetos.

# Funcionalidades principais:
# Secretária:
   - Cadastro, edição e remoção de pacientes (dados pessoais, contato, convênio)
   - Cadastro, edição e remoção de consultas (data, horário, médico, paciente, tipo)
   - Geração de relatórios de consultas do dia seguinte (com filtro por e-mail/celular)

# Médico:
   - Cadastro complementar de dados clínicos dos pacientes (hábitos, histórico de doenças, cirurgias, alergias)
   - Gerenciamento de prontuários (sintomas, diagnóstico, prescrição)
   - Geração de relatórios médicos (receita, atestado, declaração, clientes atendidos no mês)

# Gerenciador de Mensagens:
   - Envio simulado de e-mail/SMS para clientes com consultas no dia seguinte
